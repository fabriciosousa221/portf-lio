Demonstrar trabalhos feito em sala de aula, de uma forma simples e que as pessoas possam ficar entertidas com as imagens dos projetos

**Linguagens**

- HTML5
- CSS

Uma Navbar com "Inicio", "Sobre", "Projetos" e "Contatos"
